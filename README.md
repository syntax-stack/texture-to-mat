# Texture2Mat Pro for Godot 4.x

Convert multiple textures into materials instantly.

## Features
- **Batch Processing:** Select 1, 10 or 100 textures. It all works.
- **Zero Configuration:** Creates StandardMaterial3D resources with the texture assigned to the Albedo slot.

## Usage Guide
1. Select your textures in the FileSystem.
2. Ensure your textures are set to the **3D Texture** preset in the **Import** tab.
3. Go to Project -> Tools -> Create Materials from Textures.
4. Click **Create Materials from Textures**.
